About us

Welcome to Our Phone List Project

This website is a class project created for our HTML, CSS, and JavaScript course at Handassaim Technion. Our goal is to build an interactive and user-friendly phone list application that helps users manage their contacts efficiently. From adding and editing contacts to searching, sorting, and tagging them—this project is designed with both functionality and clean design in mind.

Who Are We?

We are Jobran Ayash and Majd Bathesh, two students passionate about web development and software design. Through this project, we’re applying our skills in HTML, CSS, and JavaScript to solve a practical challenge while learning real-world programming techniques and best practices.

Project Overview

Our phone list application allows users to create, edit, delete, and search through their contacts. It includes features such as image uploads, tags, favorites, responsive design, and pop-up forms. We've structured our code for clarity and maintainability and used event delegation to manage dynamic elements effectively.

Key Features

Here are some of the main features we’ve implemented in the project:

Add & Edit Contacts

Users can create new contacts and update existing ones through clean, responsive pop-up forms.

Search & Sort

Contacts can be quickly found using a search bar and sorted alphabetically for easy browsing.

Favorites

Users can mark important contacts as favorites, making them easy to find when needed.

Tagging System

Contacts can be tagged with categories like "Family", "Work", or "Friends" for better organization.

Responsive Design

The site is fully responsive and works seamlessly across desktops, tablets, and mobile devices.

Learning Experience

This project has been a valuable learning opportunity for us. We practiced DOM manipulation, form validation, file structure organization, and real user interface design—all while staying true to modern coding standards.
